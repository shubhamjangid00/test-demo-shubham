# Simple Web Application

This repository contains a simple web application for demonstration purposes. The application consists of a single HTML file displaying a basic webpage with a "Hello, World!" message.

## Technologies Used

- HTML

## Setup Instructions

**1. **Clone the repository**:
```bash
git clone <repository_url>

**2. Navigate to the project directory:
cd simple-web-app

**3. View the application:
Open index.html in a web browser to view the application.

DevOps Setup
Requirements
 1. Docker
2.Ansible
 3. Nagios
4.  Maven
 5. Jenkins
Steps
1. Docker Setup:
Install Docker on your system if not already installed.
Build a Docker image for the web application:
bash
Copy code
docker build -t simple-web-app .
Run the Docker container:
bash
Copy code
docker run -d -p 8080:80 simple-web-app
2. Ansible Setup:
Install Ansible on your system if not already installed.
Create an Ansible playbook for deploying the Docker container.
3. Nagios Setup:
Install Nagios on your system for monitoring.
Configure Nagios to monitor the web application.
4. Maven Setup:
Install Maven on your system if not already installed.
Create a Maven project for managing dependencies and building the web application.
5. Jenkins Setup:
Install Jenkins on your system if not already installed.
Create a Jenkins job for continuous integration and continuous deployment.
